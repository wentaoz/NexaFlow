# Data Ingestion Semantics References

## Source Meaning Examples

- Tableau View export: report table, but only for the visible/exportable view grain.
- Jira issue list: project evidence, not actual launch proof.
- DingTalk document: internal document evidence, not operational data unless the document itself contains data.
- Local XLSX: user-provided report data.

## Common Error

Do not state "上线于文档更新时间" unless release time is explicitly confirmed by release log or user correction.
