# Product Ops UX Patterns

## Mode language

- Quick Answer: calls AI using recent conversation and cache; does not refresh external data.
- Deep Analysis: rereads selected tables, knowledge, project evidence, external evidence, and calculation evidence.
- Concise Report: daily update with data changes, cause analysis, and actions.
- Complete Report: leadership-ready report with scope, read coverage, evidence, risks, and actions.

## Report inclusion

Include business questions about metrics, funnel, channel, risk, campaign, external event, or product performance.

Exclude questions about UI, buttons, token setup, crashes, packaging, or app debugging unless manually selected.
