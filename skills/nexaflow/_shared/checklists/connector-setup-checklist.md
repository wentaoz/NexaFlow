# Connector Setup Checklist

1. Bind the connector to the correct business space.
2. Confirm required credentials are present locally.
3. Do not paste secrets into chat, prompts, reports, logs, or exported documents.
4. Confirm read permission for the target object.
5. Confirm source type: report table, knowledge, project evidence, document evidence, or external evidence.
6. Confirm time semantics: event, created, updated, published, collected, or imported time.
7. Configure sync limits and filters.
8. Run a test sync before relying on the source in a report.
