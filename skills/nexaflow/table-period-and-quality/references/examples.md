# Table Period and Quality References

## Wide Table

Metrics in first column, periods across columns. Sort period ranges by end date.

## Long Table

Columns may include period, metric, segment, value, and percentage change. The period column is a candidate until confirmed.

## Detail Table

Multiple date columns can mean different business events: registration, approval, transaction, repayment, or event occurrence.

## Common Error

Do not default to latest-vs-previous comparison when the user asked for full-period overview.
