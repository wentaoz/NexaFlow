# Business Space Modeling References

## Input Example

`墨西哥信用卡，关注获客、申请、KYC、授信、激活、首刷、复购和还款风险。`

## Output Example

- Product type: credit card.
- Core flow: acquisition -> registration -> application -> KYC/SMS -> approval -> credit line -> card issue -> activation -> first transaction -> ongoing spend -> repayment/risk.
- Evidence domains: competition, regulation, holidays, electricity/weather, app reviews.

## Common Error

Do not copy source lists or memories from another business space unless they are explicitly global.
