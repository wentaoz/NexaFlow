## Credit Card Example

Question: `为什么最新周期交易金额下降但交易人数上涨？`

Expected path:
1. Read transaction and local-life tables.
2. Compare amount, user count, average ticket, scenario mix, card type, new/old user mix.
3. Tie upstream acquisition/KYC changes only when evidence exists.
4. Classify facts, inferences, hypotheses, and needs data.
