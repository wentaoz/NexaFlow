# NexaFlow 1.1.0

## 主要更新

- 提升 AI 分析任务的连续性：流式回答校验失败后会带着修正要求重新生成，后台延迟重试达到上限后会转为等待用户处理。
- 提升外部资料采集的可控性：Confluence、Jira、钉钉、Tableau 和搜索数据源支持重试退避，整轮资料采集会严格遵守时间预算。
- 提升大型表格导入稳定性：CSV 改用映射读取并限制异常行列规模；XLS/XLSX 可识别稀疏远端单元格、超大合并区域和异常密集工作表，避免导入时内存突增。
- 改善本地资料持续访问：数据包和本地知识文件夹保存安全作用域书签，App 重启后可继续访问用户授权的目录。

## Bug 修复

- 修复正常退出时最后一批 workspace 修改可能尚未落盘的问题；退出流程现在会等待原子写入完成。
- 增加 workspace schema 迁移和高版本只读保护，避免旧数据被静默套用新默认值，或由旧版 App 覆盖新格式数据。
- 修复钥匙串写入失败时 JSON 已清空凭据、重启后配置丢失的问题。
- 修复外部资料采集超过整轮 deadline 后仍等待单个网络请求的问题。
- 修复 AI 后台任务可无限进行延迟重试的问题。
- 修复长期运行后知识条目和纠偏消息持续增长、workspace 文件膨胀的问题。
- 修复分析资料 GUI 快照存在透明背景导致黑块的问题。

## 兼容性

- 支持 macOS 13 及以上版本。
- 安装包包含 Apple Silicon (`arm64`) 与 Intel (`x86_64`) 两种架构。
